# 🎊 لعبة الألعاب

ألعاب جماعية للجمعات والتجمعات.

## كيف تشغّلها على GitHub Pages

1. ارفع الملفات الأربعة على GitHub repository جديد
2. روح **Settings → Pages**
3. اختر **Branch: main** واضغط Save
4. بعد دقيقة يطلع لك رابط جاهز!

## الملفات

| الملف | المحتوى |
|-------|---------|
| `index.html` | هيكل الصفحة |
| `style.css` | التصميم |
| `data.js` | بيانات الألعاب |
| `app.js` | منطق الألعاب |
